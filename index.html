VitalStream | Serverless Real-Time Patient Telemetry Pipeline

Live AWS Deployment — 7 VPC PrivateLink Endpoints, HIPAA-Aligned Architecture

AWS · MLOps · Serverless · Healthcare · HIPAA

The Challenge: High-frequency physiological data (ECG at 500 Hz, sEMG at 2000 Hz) must be processed and acted on in near real-time — with zero data loss, strict PHI isolation, and clinical-grade audit trails.
The Solution: Built VitalStream, a fully serverless real-time telemetry pipeline on AWS. Devices publish via MQTT to IoT Core → Kinesis Data Streams → Lambda (DSP + HRV analysis) → SageMaker 1D-CNN + BiLSTM ensemble for anomaly detection → DynamoDB with alert routing.
Infrastructure: Terraform-provisioned 3-AZ private VPC, 7 VPC PrivateLink endpoints, customer-managed KMS CMKs, S3 Object Lock for 7-year HIPAA retention. CI/CD via GitHub Actions with security scanning and 80% coverage gate.

Laser Metal Bending | Predictive Modeling

Ensemble Regression Analysis

The Solution: Developed a comparative modeling framework using 16 different algorithms to predict deformation angles. Nanocomposite Material Research

Ultrasonic Homogenization Setup

Post-Compression Test Specimens

Stress-Strain Analysis Results

The Challenge: Improve the compressive strength of epoxy composites without significantly increasing weight. CNT epoxy composites have previously been mainly tested under tensile conditions.
The Solution: Investigated Carbon Nanotube (CNT) reinforcement strategies for both straight and helical CNTs. I developed a novel ultrasonic homogenization protocol to disperse CNTs evenly and utilized SEM/TEM microscopy to analyze the microstructural bonding.
